<h1> JavaScript Quiz</h1>

<p>
  This project is an <strong>interactive quiz built with JavaScript</strong>, HTML, and CSS.
  It allows users to answer multiple-choice questions, validate answers, display the score,
  and navigate between questions.
</p>

<hr>

<h2> Features</h2>
<ul>
  <li>Multiple-choice questions</li>
  <li>Answer validation</li>
  <li>Option selection locked after confirmation</li>
  <li>Navigation between questions (next / previous)</li>
  <li>Score tracking</li>
  <li>Simple and responsive UI</li>
  <li>Built with <strong>Vanilla JavaScript</strong> (no external libraries)</li>
</ul>

<hr>

<h2> Technologies Used</h2>
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<hr>

<ul>

  <li>Implement difficulty levels (beginner, intermediate, advanced)</li>
